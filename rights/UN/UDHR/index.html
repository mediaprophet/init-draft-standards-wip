Abstract

This document specifies an ontology for the Universal Declaration of Human Rights (UDHR), adopted by the United Nations in 1948. The ontology models the UDHR's articles as instances of a Right class, with properties describing their content, applicability, and relationships. It is expressed in OWL 2 and supports semantic web applications such as reasoning and querying.

Status of This Document

This is an unofficial ontology specification for the UDHR, created for demonstration purposes. It is not endorsed by the United Nations.

Introduction

The Universal Declaration of Human Rights (UDHR) is a foundational document outlining fundamental human rights. This ontology provides a structured, machine-readable representation of the UDHR, enabling applications in legal informatics, knowledge graphs, and semantic reasoning.

Namespace

The namespace for this ontology is http://example.org/ontology/udhr#. Prefixes used include:

  • udhr: http://example.org/ontology/udhr#
  • rdf: http://www.w3.org/1999/02/22-rdf-syntax-ns#
  • rdfs: http://www.w3.org/2000/01/rdf-schema#
  • owl: http://www.w3.org/2002/07/owl#

Ontology Structure

Classes

The ontology defines the following classes:

Class Description URI
Right Represents a human right as defined in the UDHR. udhr:Right
Person Represents an individual entitled to rights. udhr:Person
Group Represents a collective entitled to certain rights. udhr:Group

Properties

The ontology includes the following properties:

Property Type Domain Range Description
udhr:hasArticleNumber Data Property udhr:Right xsd:integer Specifies the article number of a right.
udhr:hasContent Data Property udhr:Right xsd:string Contains the textual content of a right.
udhr:appliesTo Object Property udhr:Right udhr:Person or udhr:Group Links a right to its entitled entities.

Instances

Selected articles from the UDHR are modeled as instances of udhr:Right:

Instance Article Number Content (Abbreviated)
udhr:Article1 1 All human beings are born free and equal in dignity and rights...
udhr:Article2 2 Everyone is entitled to all the rights and freedoms set forth...

Ontology Serialization

An example of the UDHR ontology in Turtle format:

@prefix udhr:  .
@prefix owl:  .
@prefix rdf:  .
@prefix rdfs:  .
@prefix xsd:  .

udhr:Right a owl:Class ;
    rdfs:label "Human Right" ;
    rdfs:comment "A right as defined in the UDHR." .

udhr:Person a owl:Class ;
    rdfs:label "Person" ;
    rdfs:comment "An individual entitled to human rights." .

udhr:hasArticleNumber a owl:DatatypeProperty ;
    rdfs:domain udhr:Right ;
    rdfs:range xsd:integer ;
    rdfs:label "has article number" .

udhr:hasContent a owl:DatatypeProperty ;
    rdfs:domain udhr:Right ;
    rdfs:range xsd:string ;
    rdfs:label "has content" .

udhr:appliesTo a owl:ObjectProperty ;
    rdfs:domain udhr:Right ;
    rdfs:range [ a owl:Class ; owl:unionOf (udhr:Person udhr:Group) ] ;
    rdfs:label "applies to" .

udhr:Article1 a udhr:Right ;
    udhr:hasArticleNumber 1 ;
    udhr:hasContent "All human beings are born free and equal in dignity and rights..." ;
    udhr:appliesTo udhr:Person .

Constraints and Axioms

The ontology enforces the following constraints:

  • Each udhr:Right must have exactly one udhr:hasArticleNumber.
  • udhr:Person and udhr:Group are disjoint classes.

Usage Examples

SPARQL query to retrieve all rights applicable to persons:

PREFIX udhr: 
SELECT ?right ?content
WHERE {
    ?right a udhr:Right ;
           udhr:hasContent ?content ;
           udhr:appliesTo udhr:Person .
}
